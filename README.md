# Modules简介
这是一个学习仓库，用于学习各类基础组件和对于本人不熟悉的C++语法进行补充学习。

# Log
希望实现一个高可用的、综合的日志库，设计目标于Note中阐述。

Log模块：
- LogConfig: 复制Log类的配置，默认从log.config中读取配置信息
- LogFormat: Log记录时的格式信息配置
